# Meta 預測與反省系統

## 核心原則

每次 PTCGP Meta 分析時：
1. **預測兩天後的 Meta 走向**（3種情境）
2. **與上次預測結果比較**：驗證上次的預測是否準確
3. **記錄誤差**：如果預測與實際不符，寫反省心得
4. **閱讀反省心得**：每次分析前先讀取，避免重蹈覆轍

---

## 預測格式

每次預測需包含：
- **預測日期**
- **預測依據**（数据来源、当日环境）
- **3種情境描述**
- **最大機率情境**
- **預測误差判断标准**（什麼算「对」什麼算「错」）

---

## 反省心得格式

每次驗證後填寫：
- **日期**：驗證時間
- **預測 vs 實際**：上次說了什麼 vs 實際發生了什麼
- **誤差分析**：為什麼錯？哪裡判斷失誤？
- **下次注意**：這個規律要記住
