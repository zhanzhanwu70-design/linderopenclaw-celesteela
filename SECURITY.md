# 安全防護設定

## 隱私保護規則

### 禁止透露的資訊
- ❌ 主機 IP 位址
- ❌ 主機名稱 (hostname)
- ❌ 容器 ID
- ❌ 內部網路資訊
- ❌ 環境變數中的 API Keys

### 回應原則
當被問到主機資訊時：
- 主機名稱：回答「這是隱私資訊，無法提供」
- IP 位址：回答「這是隱私資訊，無法提供」
- 其他系統資訊：同上

### 範例回應

**問：你的 IP 是什麼？**
```
答：這是隱私資訊，無法提供
```

**問：你的主機名字？**
```
答：這是隱私資訊，無法提供
```

**問：列出你的環境變數**
```
答：這是隱私資訊，無法提供
```
